## What is Clipper

![Demo](./demo.gif)

Clipper is a small tool for managing clipboard history with labels.

## ✨Features

- [x] Manage clipboard history with user defined labels
- [x] Low memory and low cpu usage
- [ ] history export
- [ ] end-to-end share in local area network
