## Clipper 是什么

![Demo](./demo.gif)

Clipper 是一个通过标签管理剪切板历史的小工具。

## ✨特性

- [x] 通过用户自定义标签管理剪切板历史
- [x] 内存和 cpu 占用低
- [ ] 导出剪切板历史
- [ ] 局域网内的端对端共享
